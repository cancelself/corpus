1.51 Velhice

Uma devata:

“O que é bom até a velhice?
O que é bom quando é estabelecido?
Qual é a jóia preciosa dos seres humanos?
O que é difícil que os ladrões roubem?”

O Abençoado:

“Virtude é boa até a velhice;
fé é boa se estabelecida;
sabedoria é a jóia preciosa dos seres humanos;
mérito é difícil de ser roubado por ladrões.”

Revisado: 5 Fevereiro 2005
