8.27 Poder

“Bhikkhus, há esses oito poderes. Quais oito? (1) O poder das crianças é chorar; (2) o poder das mulheres é a raiva; (3) o poder dos ladrões é uma arma; (4) o poder dos reis é a soberania; (5) o poder dos tolos é a reclamação; (6) o poder dos sábios é a inferência; (7) o poder dos estudados é a reflexão; (8) o poder dos contemplativos e brâmanes é a paciência. Esses são os oito poderes.”

5 Outubro 2013
