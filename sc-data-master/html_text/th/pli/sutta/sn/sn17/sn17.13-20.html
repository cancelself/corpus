17.13–20

๓. สุวัณณนิกขสูตร

… ดูกรภิกษุทั้งหลาย เรากำหนดรู้ใจด้วยใจแล้ว ย่อมรู้บุคคลบางคนในโลกนี้ อย่างนี้ว่า แม้เพราะแท่งทองคำเป็นเหตุ … ฯ

๔. สุวัณณนิกขสตสูตร

ดูกรภิกษุทั้งหลาย เรากำหนดใจด้วยใจแล้ว ย่อมรู้บุคคลบางคนในโลกนี้อย่างนี้ว่า แม้เพราะร้อยแห่งแท่งทองคำเป็นเหตุ … ฯ

๕. สิงคินิกขสูตร

ดูกรภิกษุทั้งหลาย เรากำหนดรู้ใจด้วยใจแล้ว ย่อมรู้บุคคลบางคนในโลกนี้อย่างนี้ว่า แม้เพราะแท่งแห่งทองสิงคิเป็นเหตุ … ฯ

๖. สิงคินิกขสตสูตร

… ดูกรภิกษุทั้งหลาย เรากำหนดรู้ใจด้วยใจแล้ว ย่อมรู้บุคคลบางคนในโลกนี้อย่างนี้ว่า แม้เพราะร้อยแท่งแห่งทองสิงคิเป็นเหตุ … ฯ

๗. ปฐวีสูตร

… ดูกรภิกษุทั้งหลาย เรากำหนดรู้ใจด้วยใจแล้ว ย่อมรู้บุคคลบางคนในโลกนี้อย่างนี้ว่า แม้เพราะแผ่นดินที่เต็มด้วยทองเป็นเหตุ … ฯ

๘. อามิสกิญจิกขสูตร

… ดูกรภิกษุทั้งหลาย เรากำหนดรู้ใจด้วยใจแล้ว ย่อมรู้บุคคลบางคนในโลกนี้อย่างนี้ว่า แม้เพราะเห็นแก่ของกำนัลเพียงเล็กน้อยเป็นเหตุ … ฯ

๙. ชีวิตสูตร

… ดูกรภิกษุทั้งหลาย เรากำหนดรู้ใจด้วยใจแล้ว ย่อมรู้บุคคลบางคนในโลกนี้อย่างนี้ว่า แม้เพราะเหตุแห่งชีวิตเป็นเหตุ … ฯ

๑๐. ชนปทกัลยาณีสูตร

… ดูกรภิกษุทั้งหลาย เรากำหนดรู้ใจด้วยใจแล้ว ย่อมรู้บุคคลบางคนในโลกนี้อย่างนี้ว่า แม้เพราะนางงามประจำชนบทเป็นเหตุ ท่านผู้นี้ก็ไม่จงใจพูดมุสา แต่สมัยต่อมา เราเห็นเขาถูก ลาภ สักการะและชื่อเสียงครอบงำย่ำยีจิตแล้ว ก็กล่าวมุสาทั้งที่รู้ได้ ดูกรภิกษุทั้งหลาย ลาภ สักการะ และชื่อเสียง ทารุณ ฯลฯอย่างนี้แล เธอทั้งหลายพึงศึกษาอย่างนี้แหละ ฯ

จบสูตรที่ ๑๐

จบทุติยวรรคที่ ๒

รวมพระสูตรที่มีในวรรคนี้ คือ ๑. สุวัณณปาติสูตร ๒. รูปิยปาติสูตร ๓. สุวัณณนิกขสูตร ๔. สุวัณณนิกขสตสูตร ๕. สิงคินิกขสูตร ๖. สังคินิกขสตสูตร ๗. ปฐวีสูตร ๘. อามิสกิญจิกขสูตร 9. ชีวิตสูตร ๑๐. ชนปทกัลยาณีสูตร ฯ
