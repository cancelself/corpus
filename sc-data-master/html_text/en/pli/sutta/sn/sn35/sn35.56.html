35.56–59. Abandoning the Taints, Etc.

SC 1… “Venerable sir, how should one know, how should one see, for the taints to be abandoned?… for the taints to be uprooted?… for the underlying tendencies to be abandoned?… for the underlying tendencies to be uprooted?”

SC 2“Bhikkhu, when one knows and sees the eye as nonself, the underlying tendencies are uprooted. When one knows and sees forms as nonself … all as above … When one knows and sees thus, bhikkhu, the underlying tendencies are uprooted.”
