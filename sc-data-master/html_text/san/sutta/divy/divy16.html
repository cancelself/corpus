16. śukapotakāvadānam

Vai 123.001 śrāvastyāṁ nidānam |
Vai 123.002 tena khalu samayena anāthapiṇḍadena gṛhaptinā dvau śukaśāvakau pratilabdhau |
Vai 123.003 tena niveśanaṁ nītvā ālāpitau poṣitau saṁvardhitau mānuṣālāpaṁ ca śikṣāpitau |
Vai 123.004 tayoścāyuṣmānānando ’bhīkṣṇamāgatya caturāryasatyasamprativedhikīṁ dharmadeśanāṁ karoti—yaduta idaṁ duḥkham, ayaṁ duḥkhasamudayaḥ, ayaṁ duḥkhanirodhaḥ, iyaṁ duḥkhanirodhagāminī pratipaditi |
Vai 123.005 sthavirasthavirā api bhikṣavo ’nāthapiṇḍadasya gṛhapater niveśanamupasaṁkrāmanti śāriputramaudgalyāyanakāśyapānandaraivataprabhṛtayaḥ |
Vai 123.007 teṣāṁ kālānukālamupasaṁkrāmatāṁ tābhyāṁ śukaśāvakābhyāṁ nāmāni parijñātāni |
Vai 123.008 yāvadapareṇa samayenāyuṣmāñ śāriputro ’nāthapiṇḍadasya gṛhapater niveśanamanuprāptaḥ |
Vai 123.011 adrāṣṭāṁ tau śukaśāvakau āyuṣmantaṁ śāriputram |
Vai 123.009 dṛṣṭvā antarjanamāmantrayatah—eṣa bhadantaḥ sthaviraḥ śāriputra āgacchati, āsanamasya prajñāpayateti |
Vai 123.010 evamāyuṣmantaṁ mahāmaudgalyāyanaṁ kāśyapaṁ raivatamāyuṣmantamānandaṁ dṛṣṭvā kathayatah—eṣo ’smākamācāryānanda āgacchati, āsanamasya prajñāpayateti |
Vai 123.012 yāvadapareṇa samayena bhagavānanāthapiṇḍadasya gṛhapater niveśanamanuprāptaḥ |
adrāṣṭāṁ tau śukaśāvakau bhagavantaṁ dūrādevāgacchantaṁ prāsādikaṁ prasādanīyaṁ śāntendriyaṁ śāntamānasaṁ parameṇa cittamatyupaśamena samanvāgataṁ suvarṇayūpamiva śriyā jvalantam |
Vai 123.014 dṛṣṭvā ca punastvaritatvaritamantarjanamāmantrayatah—eṣa bhadanto bhagavānāgacchati, āsanamasya prajñāpayateti hṛṣṭamadhurasvareṇa nikūjataḥ |
Vai 123.016 atha bhagavāṁstayoranugrahārthaṁ praviśya prajñapta evāsane niṣaṇṇaḥ |
niṣadya bhagavatā śukaśāvakau caturāryasatyasamprativedhikayā dharmadeśanayā śaraṇagamanaśikṣāpadeṣu pratiṣṭhāpitau |
Vai 123.018 atha bhagavāñ śukaśāvakau antarjanaṁ ca dharmyayā kathayā saṁdarśya samādāpya samuttejya saṁpraharṣyotthāyāsanāt prakrāntaḥ |
Vai 123.019 tau cāntarjanasya viharataḥ pramādavihāriṇau biḍālena prāṇinā gṛhītau |
Vai 123.020 vihvalavadanau cchidyamāneṣu marmasu mucyamāneṣu saṁdhiṣu namo buddhāya, namo dharmāya, namaḥ saṁghāyetyuktvā kālagatau cāturmahārājakāyikeṣu deveṣūpapannau |
|
Vai 123.022 atha bhagavānanyatamasmin pradeśe smitamakārṣīt |
adrākṣīdāyuṣmānānando bhagavantaṁ smitaṁ prāviṣkurvantam |
Vai 123.023 dṛṣṭvā ca punarbhagavantamidamavocat—nāhetupratyayaṁ bhadanta tathāgatā arhantaḥ samyaksambuddhāḥ smitaṁ prāviṣkurvanti |
Vai 123.024 ko bhadanta hetuḥ kaḥ pratyayaḥ smitasya prāviṣkaraṇe? evametadānanda, evametat |
Vai 123.025 nāhetupratyayaṁ tathāgatā arhantaḥ samyaksambuddhāḥ smitaṁ prāviṣkurvanti |
Vai 123.026 dṛṣṭau tvayā ānanda tau śukaśāvakau? dṛṣṭau bhadanta |
tau ānanda śukaśāvakau mama samanantaraprakrāntasya biḍālena prāṇinā jīvitād vyaparopitau |
Vai 123.027 tau buddhadharmasaṁghāvalambanayā smṛtyā kālagatau cāturmahārājakāyikeṣu deveṣūpapannau |
|
Vai 123.029 atha saṁbahulā bhikṣavaḥ pūrvāhṇe nivāsya pātracīvaramādāya śrāvastīṁ piṇḍāya prāvikṣan |
Vai 123.030 aśrauṣuḥ saṁbahulā bhikṣavaḥ śrāvastīṁ piṇḍāya pracaranto ’nāthapiṇḍadasya gṛhapater niveśane śukaśāvakau—namo buddhāya, namo dharmāya, namaḥ saṁghāyeti biḍālena prāṇinā vyaparopitauVai 124.000 iti | Vai 124.001

śrutvā ca punaḥ śrāvastīṁ piṇḍāya caritvā kṛtabhaktakṛtyāḥ paścādbhaktapiṇḍapātapratikrāntāḥ pātracīvaraṁ pratisāmayya pādau prakṣālya yena bhagavāṁstenopasaṁkrāntāḥ |
Vai 124.002 upasaṁkramya bhagavataḥ pādau śirasā vanditvā ekānte niṣaṇṇāḥ |
Vai 124.003 ekāntaniṣaṇṇāḥ saṁbahulā bhikṣavo bhagavantamidamavocan—iha vayaṁ bhadanta saṁbahulā bhikṣavaḥ pūrvavad yāvadanāthapiṇḍadasya gṛhapater niveśane dvau śukaśāvakau—namo buddhāya, namo dharmāya, namaḥ saṁghāyeti kurvāṇau biḍālena prāṇinā jīvitādvyaparopitau iti |
Vai 124.006 tayorbhadanta kā gatiḥ, kopapattiḥ, ko ’bhisamparāyah? bhagavānāha—tau bhikṣavaḥ śukaśāvakau tasya śaraṇagamanasya vipākena ṣaṭtriṁśatkṛtvaścāturmahārājakāyikeṣu deveṣūpapatsyete, ṣaṭtriṁśatkṛtvastrāyastriṁśeṣu, yāmeṣu, tuṣiteṣu, nirmāṇaratiṣu, paranirmitavaśavartiṣu deveṣūpapastyete |
124.009 tatastāvat ṣaṭsu kāmāvacareṣu deveṣu sattvā vyapasaṁsṛtya paścime bhave paścime nikete paścime ātmabhāvapratilambhe manuṣyapratilābhaṁ labdhvā pratyekāṁ bodhimabhisambhotsyete, dharmaśca sudharmaśca pratyekabuddhau bhaviṣyataḥ |
Vai 124.011 evaṁ hi bhikṣavo mahāphalaṁ dharmaśravaṇaṁ mahānuśaṁsakam, kaḥ punarvādo dharmadeśanā dharmābhisamayo vā |
Vai 124.012 tasmāttarhi bhikṣava evaṁ śikṣitavyam yanno dharmaśravaṇābhiratā bhaviṣyāmaḥ |
Vai 124.013 ityevaṁ vo bhikṣavaḥ śikṣitavyam |
|
Vai 124.014 idamavocadbhagavān |
ātamanasaste bhikṣavo bhagavato bhāṣitamabhyanandan |
|
Vai 124.015 iti śrīdivyāvadāne śukapotakāvadānaṁ ṣoḍaśam |
|
