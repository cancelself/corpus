1.82–97. Dovenskap

82

SC 1Jeg vet ikke noen annen enkeltfaktor som fører så store ulemper med seg som dovenskap, munker. Dovenskap fører med seg store ulemper.

83

SC 2Jeg vet ikke noen annen enkeltfaktor som fører så store fordeler med seg som flid, munker. Flid fører med seg store fordeler.

84

SC 3Jeg vet ikke noen annen enkeltfaktor som fører så store ulemper med seg som latskap, munker. Latskap fører med seg store ulemper.

85

SC 4Jeg vet ikke noen annen enkeltfaktor som fører så store fordeler med seg som iherdig innsats, munker. Iherdig innsats fører med seg store fordeler.

86

SC 5Jeg vet ikke noen annen enkeltfaktor som fører så store ulemper med seg som det å være kravstor, munker. Det å være kravstor fører med seg store ulemper.

87

SC 6Jeg vet ikke noen annen enkeltfaktor som fører så store fordeler med seg som nøysomhet, munker. Nøysomhet fører med seg store fordeler.

88

SC 7Jeg vet ikke noen annen enkeltfaktor som fører så store ulemper med seg som misnøye, munker. Misnøye fører med seg store ulemper.

89

SC 8Jeg vet ikke noen annen enkeltfaktor som fører så store fordeler med seg som tilfredshet, munker. Tilfredshet fører med seg store fordeler.

90

SC 9Jeg vet ikke noen annen enkeltfaktor som fører så store ulemper med seg som ulogisk og usystematisk tenkning, munker. Ulogisk og usystematisk tenkning fører med seg store ulemper.

91

SC 10Jeg vet ikke noen annen enkeltfaktor som fører så store fordeler med seg som logisk og systematisk tenkning, munker. Logisk og systematisk tenkning fører med seg store fordeler.

92

SC 11Jeg vet ikke noen annen enkeltfaktor som fører så store ulemper med seg som manglende forståelse, munker. Manglende forståelse fører med seg store ulemper.

93

SC 12Jeg vet ikke noen annen enkeltfaktor som fører så store fordeler med seg som logisk og systematisk tenkning, munker. Logisk og systematisk tenkning fører med seg store fordeler.

94

SC 13Jeg vet ikke noen annen enkeltfaktor som fører så store ulemper med seg som dårlig vennskap, munker. Dårlig vennskap fører med seg store ulemper.

95

SC 14Jeg vet ikke noen annen enkeltfaktor som fører så store fordeler med seg som godt vennskap, munker. Godt vennskap fører med seg store fordeler.

96

SC 15Jeg vet ikke noen annen enkeltfaktor som fører så store ulemper med seg som det å være opptatt av usunne sinnstilstander og ikke opptatt av sunne sinnstilstander, munker. Det å være opptatt av usunne sinnstilstander og ikke opptatt av sunne sinnstilstander fører med seg store ulemper.

97

SC 16Jeg vet ikke noen annen enkeltfaktor som fører så store fordeler med seg som det å være opptatt av sunne sinnstilstander og ikke opptatt av usunne sinnstilstander, munker. Det å være opptatt av sunne sinnstilstander og ikke opptatt av usunne sinnstilstander fører med seg store fordeler.’
