1.616–627. Det som ikke dør

616

SC 1De som ikke tar del i oppmerksomhet på kroppen, tar heller ikke del i det som ikke dør, munker. De som tar del i oppmerksomhet på kroppen, tar del i det som ikke dør.

617

SC 2De som ikke har tatt del i oppmerksomhet på kroppen, har heller tatt ikke del i det som ikke dør, munker. De som har tatt del i oppmerksomhet på kroppen, har tatt del i det som ikke dør.

618

SC 3Det som ikke dør, forsvinner for dem som oppmerksomhet på kroppen forsvinner for, munker. Det som ikke dør, forsvinner ikke for dem som oppmerksomhet på kroppen ikke forsvinner for.

619

SC 4Det som ikke dør, går tapt for dem som oppmerksomhet på kroppen går tapt for, munker. Det som ikke dør, går ikke tapt for dem som oppmerksomhet på kroppen ikke går tapt for.

620

SC 5De som neglisjerer oppmerksomhet på kroppen, neglisjerer det som ikke dør, munker. De som ikke neglisjerer oppmerksomhet på kroppen, neglisjerer ikke det som ikke dør.

621

SC 6De som glemmer oppmerksomhet på kroppen, glemmer det som ikke dør, munker. De som ikke glemmer oppmerksomhet på kroppen, glemmer ikke det som ikke dør.

622

SC 7De som ikke praktiserer oppmerksomhet på kroppen, praktiserer ikke det som ikke dør, munker. De som praktiserer oppmerksomhet på kroppen, praktiserer det som ikke dør.

623

SC 8De som ikke utvikler oppmerksomhet på kroppen, utvikler ikke det som ikke dør, munker. De som utvikler oppmerksomhet på kroppen, utvikler det som ikke dør.

624

SC 9De som ikke gjør mye ut av oppmerksomhet på kroppen, gjør ikke mye ut av det som ikke dør, munker. De som gjør mye ut av oppmerksomhet på kroppen, gjør mye ut av det som ikke dør.

625

SC 10De som ikke forstår oppmerksomhet på kroppen, forstår ikke det som ikke dør, munker. De som forstår oppmerksomhet på kroppen, forstår det som ikke dør.

626

SC 11De som ikke har full forståelse av oppmerksomhet på kroppen, har ikke full forståelse av det som ikke dør, munker. De som har full forståelse av oppmerksomhet på kroppen, har full forståelse av det som ikke dør.

627

SC 12De som ikke realiserer oppmerksomhet på kroppen, realiserer ikke det som ikke dør, munker. De som realiserer oppmerksomhet på kroppen, realiserer det som ikke dør.
