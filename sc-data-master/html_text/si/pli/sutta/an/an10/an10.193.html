  • 4. චතුත්‍ථ පණ්ණාසකය
  • (19) 4. අරියමග්ග වර්‍ගය

10.193. උප්පාදෙතබ්බ ධම්ම සූත්‍රය

Msdiv 193’’මහණෙනි, තොපට උත්පාදයිතව්‍ය ධර්‍ම හා අනුත්පාදයිතව්‍ය ධර්‍ම දෙසන්නෙමි. එය හොඳට අසව්. මෙනෙහි කරවු. කියන්නෙමි.’’ ’’ස්වාමීනි, එසේය’’ යි ඒ භික්‍ෂූහු භාග්‍යවතුන් වහන්සේට පිළිතුරු දුන්හ. භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක.’’

’’මහණෙනි, උත්පාදයිතව්‍ය ධර්‍ම කවරේද? සතුන් මැරීම, හොරකම, කාමයන්හි වැරදි හැසිරීම, බොරුකීම, කේළාම් කීම, ඵරුෂ වචන කීම, හිස් වචන කීම, දැඩි ලෝභකම, ක්‍රෝධය, වැරදි දැකීමයි. මහණෙනි, මෙය උත්පාදයිතව්‍ය ධර්‍මයයි කියනු ලැබේ.’’

’’මහණෙනි, අනුත්පාදයිතව්‍ය ධර්‍ම කවරේද? සතුන් මැරීමෙන් වෙන්වීම, හොරකමින් වෙන්වීම, කාමයන්හි වැරදි හැසිරීමෙන් වෙන්වීම, බොරුකීමෙන් වෙන්වීම, කේළාම් කීමෙන් වෙන්වීම, ඵරුෂ වචන කීමෙන් වෙන්වීම, හිස් වචන කීමෙන් වෙන්වීම, තද ලෝභකම නොවීම, ක්‍රෝධය නොවීම, නිවැරදි දැකීමයි. මහණෙනි, මෙය අනුත්පාදයිතව්‍ය ධර්‍මයයි කියනු ලැබේ.’’

’’මහණෙනි, මේ ධර්‍ම දශයෙන් යුක්තවූ පුද්ගල තෙම සේවනය කළ යුතුයි.’’
