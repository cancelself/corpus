• Aṅguttara Nikāya

3.47. Fabriqué

«Ô Bhikkhus, il y a trois caractéristiques fabriquées qui caractérisent ce qui est fabriqué. Quelles sont-elles? L'apparition est perceptible, la disparition est perceptible, l'altération (littéralement, la différence) pendant l'existence est perceptible.

«Voici les trois caractéristiques fabriquées qui caractérisent ce qui est fabriqué.

«Maintenant, il y a trois caractéristiques non-fabriquées qui caractérisent ce qui est non-fabriqué. Quelles sont-elles? Aucune apparition n'est perceptible, aucune disparition n'est perceptible, aucune altération (littéralement, la différence) pendant l'existence n'est perceptible.

«Voici les trois caractéristiques non-fabriquées qui caractérisent ce qui est non-fabriqué.»

[Autre traduction possible:]

«Ô Bhikkhus, il y a trois caractéristiques fabriquées qui caractérisent ce qui est fabriqué. Quelles sont-elles? L'apparition est perceptible, la disparition est perceptible, l'altération (littéralement, la différence) de ce qui existe est perceptible.

«Voici les trois caractéristiques fabriquées qui caractérisent ce qui est fabriqué.

«Maintenant, il y a trois caractéristiques non-fabriquées qui caractérisent ce qui est non-fabriqué. Quelles sont-elles? Aucune apparition n'est perceptible, aucune disparition n'est perceptible, aucune altération (littéralement, la différence) de ce qui existe n'est perceptible.

«Voici les trois caractéristiques non-fabriquées qui caractérisent ce qui est non-fabriqué.»
