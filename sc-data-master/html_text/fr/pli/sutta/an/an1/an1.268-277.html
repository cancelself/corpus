• Aṅguttara Nikāya

Impossible

1.268

Vacana 268 Il est impossible, bhikkhus, impensable qu’une personne accomplie en vue [correcte] envisage un quelconque phénomène construit comme étant permanent. Cela est impossible. Mais il est possible, bhikkhus, qu’un individu ordinaire envisage un certain phénomène construit comme étant permanent. Cela est possible.

1.269

Vacana 269 Il est impossible, bhikkhus, impensable qu’une personne accomplie en vue [correcte] envisage un quelconque phénomène construit comme étant agréable. Cela est impossible. Mais il est possible, bhikkhus, qu’un individu ordinaire envisage un certain phénomène construit comme étant agréable. Cela est possible.

1.270

Vacana 270 Il est impossible, bhikkhus, impensable qu’une personne accomplie en vue [correcte] envisage un quelconque phénomène comme étant soi. Cela est impossible. Mais il est possible, bhikkhus, qu’un individu ordinaire envisage un certain phénomène comme étant soi. Cela est possible.

1.271

Vacana 271 Il est impossible, bhikkhus, impensable qu’une personne accomplie en vue [correcte] prenne la vie de sa mère. Cela est impossible. Mais il est possible, bhikkhus, qu’un individu ordinaire prenne la vie de sa mère. Cela est possible.

1.272

Vacana 272 Il est impossible, bhikkhus, impensable qu’une personne accomplie en vue [correcte] prenne la vie de son père. Cela est impossible. Mais il est possible, bhikkhus, qu’un individu ordinaire prenne la vie de son père. Cela est possible.

1.273

Vacana 273 Il est impossible, bhikkhus, impensable qu’une personne accomplie en vue [correcte] prenne la vie d’un arhat. Cela est impossible. Mais il est possible, bhikkhus, qu’un individu ordinaire prenne la vie d’un arhat. Cela est possible.

1.274

Vacana 274 Il est impossible, bhikkhus, impensable qu’une personne accomplie en vue [correcte] fasse saigner le Tathagata avec un esprit perverti. Cela est impossible. Mais il est possible, bhikkhus, qu’un individu ordinaire fasse saigner le Tathagata avec un esprit perverti. Cela est possible.

1.275

Vacana 275 Il est impossible, bhikkhus, impensable qu’une personne accomplie en vue [correcte] crée une scission du Sangha. Cela est impossible. Mais il est possible, bhikkhus, qu’un individu ordinaire crée une scission du Sangha. Cela est possible.

1.276

Vacana 276 Il est impossible, bhikkhus, impensable qu’une personne accomplie en vue [correcte] choisisse un autre enseignant [que le Bouddha]. Cela est impossible. Mais il est possible, bhikkhus, qu’un individu ordinaire choisisse un autre enseignant [que le Bouddha]. Cela est possible.

1.277

Vacana 277 Il est impossible, bhikkhus, impensable que deux arhats correctement et pleinement éveillés apparaissent simultanément dans le même système-monde. Cela est impossible. Mais il est possible, bhikkhus, qu’un seul arhat correctement et pleinement éveillé apparaisse dans un système-monde. Cela est possible.
