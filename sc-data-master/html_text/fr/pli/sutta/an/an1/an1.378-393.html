• Aṅguttara Nikāya

Répétitions sur une acquisition

1.378

Vacana 378 Véritablement, bhikkhus, ceci est une acquisition : être quelqu’un qui séjourne en forêt.

1.379

Vacana 379 Véritablement, bhikkhus, ceci est une acquisition : être quelqu’un qui se nourrit d’aumônes.

1.380

Vacana 380 Véritablement, bhikkhus, ceci est une acquisition : être quelqu’un qui porte des haillons abandonnés comme déchets.

1.381

Vacana 381 Véritablement, bhikkhus, ceci est une acquisition : être quelqu’un qui porte seulement trois robes.

1.382

Vacana 382 Véritablement, bhikkhus, ceci est une acquisition : être quelqu’un qui fait des discours sur le Dhamma.

1.383

Vacana 383 Véritablement, bhikkhus, ceci est une acquisition : être quelqu’un qui connaît la Discipline par cœur.

1.384

Vacana 384 Véritablement, bhikkhus, ceci est une acquisition : être quelqu’un de très instruit.

1.385

Vacana 385 Véritablement, bhikkhus, ceci est une acquisition : être un ancien.

1.386

Vacana 386 Véritablement, bhikkhus, ceci est une acquisition : avoir bonne allure.

1.387

Vacana 387 Non traduit.

1.388

Vacana 388 Non traduit.

1.389

Vacana 389 Véritablement, bhikkhus, ceci est une acquisition : être d’une bonne famille.

1.390

Vacana 390 Véritablement, bhikkhus, ceci est une acquisition : avoir belle allure.

1.391

Vacana 391 Véritablement, bhikkhus, ceci est une acquisition : être un bon orateur.

1.392

Vacana 392 Véritablement, bhikkhus, ceci est une acquisition : avoir peu de désirs.

1.393

Vacana 393 Véritablement, bhikkhus, ceci est une acquisition : être en bonne santé.
