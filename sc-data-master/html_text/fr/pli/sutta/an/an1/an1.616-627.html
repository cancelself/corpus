• Aṅguttara Nikāya

Le Sans-Mort

1.616

Vacana 616 Ils ne profitent pas du Sans-Mort, bhikkhus, ceux qui ne profitent pas de la présence d’esprit dirigée vers le corps. Ils profitent du Sans-Mort, bhikkhus, ceux qui profitent de la présence d’esprit dirigée vers le corps.

1.617

Vacana 617 Ils n’ont pas profité du Sans-Mort, bhikkhus, ceux qui n’ont pas profité de la présence d’esprit dirigée vers le corps. Ils ont profité du Sans-Mort, bhikkhus, ceux qui ont profité de la présence d’esprit dirigée vers le corps.

1.618

Vacana 618 Ils ont manqué le Sans-Mort, bhikkhus, ceux qui ont manqué la présence d’esprit dirigée vers le corps. Ils n’ont pas manqué le Sans-Mort, bhikkhus, ceux qui n’ont pas manqué la présence d’esprit dirigée vers le corps.

1.619

Vacana 619 Ils ont perdu le Sans-Mort, bhikkhus, ceux qui ont perdu la présence d’esprit dirigée vers le corps. Ils n’ont pas perdu le Sans-Mort, bhikkhus, ceux qui n’ont pas perdu la présence d’esprit dirigée vers le corps.

1.620

Vacana 620 Ils ont négligé le Sans-Mort, bhikkhus, ceux qui ont négligé la présence d’esprit dirigée vers le corps. Ils n’ont pas négligé le Sans-Mort, bhikkhus, ceux qui n’ont pas négligé la présence d’esprit dirigée vers le corps.

1.621

Vacana 621 Ils ont oublié le Sans-Mort, bhikkhus, ceux qui ont oublié la présence d’esprit dirigée vers le corps. Ils n’ont pas oublié le Sans-Mort, bhikkhus, ceux qui n’ont pas oublié la présence d’esprit dirigée vers le corps.

1.622

Vacana 622 Ils n’ont pas poursuivi le Sans-Mort, bhikkhus, ceux qui n’ont pas poursuivi la présence d’esprit dirigée vers le corps. Ils ont poursuivi le Sans-Mort, bhikkhus, ceux qui ont poursuivi la présence d’esprit dirigée vers le corps.

1.623

Vacana 623 Ils n’ont pas développé le Sans-Mort, bhikkhus, ceux qui n’ont pas développé la présence d’esprit dirigée vers le corps. Ils ont développé le Sans-Mort, bhikkhus, ceux qui ont développé la présence d’esprit dirigée vers le corps.

1.624

Vacana 624 Ils n’ont pas pratiqué fréquemment pour le Sans-Mort, bhikkhus, ceux qui n’ont pas pratiqué fréquemment la présence d’esprit dirigée vers le corps. Ils ont pratiqué fréquemment pour le Sans-Mort, bhikkhus, ceux qui ont pratiqué fréquemment la présence d’esprit dirigée vers le corps.

1.625

Vacana 625 Ils n’ont pas connu directement le Sans-Mort, bhikkhus, ceux qui n’ont pas connu directement la présence d’esprit dirigée vers le corps. Ils ont connu directement le Sans-Mort, bhikkhus, ceux qui ont connu directement la présence d’esprit dirigée vers le corps.

1.626

Vacana 626 Ils n’ont pas compris complètement le Sans-Mort, bhikkhus, ceux qui n’ont pas compris complètement la présence d’esprit dirigée vers le corps. Ils ont compris complètement le Sans-Mort, bhikkhus, ceux qui ont compris complètement la présence d’esprit dirigée vers le corps.

1.627

Vacana 627 Ils n’ont pas atteint le Sans-Mort, bhikkhus, ceux qui n’ont pas atteint la présence d’esprit dirigée vers le corps. Ils ont atteint le Sans-Mort, bhikkhus, ceux qui ont atteint la présence d’esprit dirigée vers le corps.
