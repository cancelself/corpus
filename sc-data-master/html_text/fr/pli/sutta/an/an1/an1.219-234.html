• Aṅguttara Nikāya

Les disciples les plus distingués (4)

1.219

Vacana 219 Bhikkhus, le plus distingué parmi mes disciples bhikkhus très instruits est Ananda.

1.220

Vacana 220 Bhikkhus, le plus distingué parmi mes disciples bhikkhus ayant une bonne mémoire est Ananda.

1.221

Vacana 221 Bhikkhus, le plus distingué parmi mes disciples bhikkhus apprenant rapidement est Ananda.

1.222

Vacana 222 Bhikkhus, le plus distingué parmi mes disciples bhikkhus résolus [à apprendre par cœur] est Ananda.

1.223

Vacana 223 Bhikkhus, le plus distingué parmi mes disciples bhikkhus ayant été assistants personnels est Ananda.

1.224

Vacana 224 Bhikkhus, le plus distingué parmi mes disciples bhikkhus ayant une grande assemblée [de dévots] est Ourouvélakassapa.

1.225

Vacana 225 Bhikkhus, le plus distingué parmi mes disciples bhikkhus inspirant la confiance aux familles est Kaloudayi.

1.226

Vacana 226 Bhikkhus, le plus distingué parmi mes disciples bhikkhus en bonne santé est Bakoula (MN 124).

1.227

Vacana 227 Bhikkhus, le plus distingué parmi mes disciples bhikkhus se remémorant leurs existences passées est Sobhita.

1.228

Vacana 228 Bhikkhus, le plus distingué parmi mes disciples bhikkhus connaissant par cœur la Discipline est Oupali.

1.229

Vacana 229 Bhikkhus, le plus distingué parmi mes disciples bhikkhus exhortant les bhikkhunis est Nandaka (MN 146).

1.230

Vacana 230 Bhikkhus, le plus distingué parmi mes disciples bhikkhus vigilants à l’entrée des facultés sensorielles est Nanda (AN 8.9).

1.231

Vacana 231 Bhikkhus, le plus distingué parmi mes disciples bhikkhus exhortant les bhikkhus est Mahakappina (SN 21.11, SN 54.7).

1.232

Vacana 232 Bhikkhus, le plus distingué parmi mes disciples bhikkhus habiles avec l’élément feu est Sagata (Pc 51).

1.233

Vacana 233 Bhikkhus, le plus distingué parmi mes disciples bhikkhus recevant des discours éloquents est Radha (SN 22.71, SN 23, SN 35.76 à 78).
