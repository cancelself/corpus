• Aṅguttara Nikāya

La vie brahmique

Bhikkhus, cette vie brahmique n'est pas vécue pour faire plaisir aux gens, ni pour les flatter, ni pour [obtenir] acquisitions, honneurs et renommée, ni pour gagner dans des débats, ni [en pensant:] "Que les gens sachent que je suis ainsi!" Mais cette vie brahmique, bhikkhus, est vécue pour la restreinte, pour l'abandon, pour le désintéressement, pour la cessation.
