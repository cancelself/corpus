• Aṅguttara Nikāya

Ce qui réduit la longévité

125. Bhikkhus, ces cinq choses réduisent la longévité. Quelles sont ces cinq? Faire ce qui est nuisible, ne pas savoir se modérer dans ce qui est bénéfique, se nourrir de ce qui est inapproprié, sortir au mauvais moment, et le non-célibat. Ces cinq choses, bhikkhus, réduisent la longévité.

Bhikkhus, ces cinq choses augmentent la longévité. Quelles sont ces cinq? Faire ce qui est bénéfique, savoir se modérer dans ce qui est bénéfique, se nourrir de ce qui est approprié, sortir au bon moment, et le célibat. Ces cinq choses, bhikkhus, augmentent la longévité.

126. Bhikkhus, ces cinq choses réduisent la longévité. Quelles sont ces cinq? Faire ce qui est nuisible, ne pas savoir se modérer dans ce qui est bénéfique, se nourrir de ce qui est inapproprié, manquer de vertu, et avoir de mauvais amis. Ces cinq choses, bhikkhus, réduisent la longévité.

Bhikkhus, ces cinq choses augmentent la longévité. Quelles sont ces cinq? Faire ce qui est bénéfique, savoir se modérer dans ce qui est bénéfique, se nourrir de ce qui est approprié, être vertueux, et avoir des amis bénéfiques. Ces cinq choses, bhikkhus, augmentent la longévité.
