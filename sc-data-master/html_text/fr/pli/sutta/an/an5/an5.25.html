• Aṅguttara Nikāya

Protégée

Lorsque, bhikkhus, la vue correcte est protégée par cinq facteurs, elle a la libération de l'esprit pour fruit, elle a la libération de l'esprit pour rétribution, elle a la libération par le discernement pour fruit, elle a la libération par le discernement pour rétribution. Quels sont ces cinq?

En cela, bhikkhus, la vue correcte est protégée par la vertu, elle est protégée par l'instruction, elle est protégée par les discussions, elle est protégée par la Tranquillité, elle est protégée par la Vision discernante.

Lorsque, bhikkhus, la vue correcte est protégée par ces cinq facteurs, elle a la libération de l'esprit pour fruit, elle a la libération de l'esprit pour rétribution, elle a la libération par le discernement pour fruit, elle a la libération par le discernement pour rétribution.
