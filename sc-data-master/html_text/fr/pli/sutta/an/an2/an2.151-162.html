2.151–162. Accueil

2.151

Vacana 151Il y a, bhikkhus, ces deux [types d']accueil. Quels sont ces deux? L'accueil avec les choses matérielles et l'accueil avec le Dhamma. Voici, bhikkhus, quels sont ces deux [types d']accueil. De ces deux [types d']accueil, le plus éminent est l'accueil avec le Dhamma.

2.152

Vacana 152Il y a, bhikkhus, ces deux [types de] service. Quels sont ces deux? Le service dans les choses matérielles et le service dans le Dhamma. Voici, bhikkhus, quels sont ces deux [types de] service. De ces deux [types de] service, le plus éminent est le service dans le Dhamma.
