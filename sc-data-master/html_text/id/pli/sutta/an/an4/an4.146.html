4.146. Waktu (1)

“Ada, para bhikkhu, empat waktu ini. Apakah empat ini? Waktu untuk mendengarkan Dhamma, waktu untuk mendiskusikan Dhamma, waktu untuk ketenangan, dan waktu untuk pandangan terang. Ini adalah keempat waktu itu.
