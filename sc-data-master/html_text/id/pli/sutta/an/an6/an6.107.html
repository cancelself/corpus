6.107. Nafsu

“Para bhikkhu, ada tiga hal ini. Apakah tiga ini? (1) Nafsu, (2) kebencian, dan (3) delusi. Ini adalah ketiga hal itu. Tiga hal [lainnya] harus dikembangkan untuk meninggalkan ketiga hal ini. Apakah tiga ini? PTS vp Pali 446 (4) Ketidak-menarikan harus dikembangkan untuk meninggalkan nafsu. (5) Cinta-kasih harus dikembangkan untuk meninggalkan kebencian. (6) Kebijaksanaan harus dikembangkan untuk meninggalkan delusi. Ketiga hal ini harus dikembangkan untuk meninggalkan ketiga hal sebelumnya.”

Translated by Indra Anggara for Dhamma Citta.
