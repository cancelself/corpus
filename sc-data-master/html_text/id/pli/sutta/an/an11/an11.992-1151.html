• Aṅguttara Nikāya

11.992–1151

“Para bhikkhu, demi pengetahuan langsung … demi pemahaman penuh … demi kehancuran sepenuhnya … demi meninggalkan … demi hancurnya … demi hilangnya … demi peluruhan … demi lenyapnya … demi berhentinya … demi terlepasnya kebencian … delusi … kemarahan … permusuhan … sikap merendahkan … sikap kurang ajar … iri … kekikiran … kecurangan … muslihat … kekeras-kepalaan … sikap berapi-api … keangkuhan … kesombongan … kemabukan … kelengahan … maka kesebelas hal ini harus dikembangkan.”

Ini adalah apa yang dikatakan oleh Sang Bhagavā. Dengan gembira, para bhikkhu itu puas mendengar pernyataan Sang Bhagavā.
