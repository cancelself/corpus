8.91–117

Umat awam perempuan Bojjhā … Sirimā … Padumā … Sutanā … Manujā … Uttarā … Muttā … Khemā … Somā … Ruci … Putri Cundī … umat awam perempuan Bimbī … Putri Sumanā … PTS vp Pali 348 Ratu Mallikā … umat awam perempuan Tissā … Soṇā ibunya Tissā … ibunya Soṇā … Kāṇā … ibunya Kāṇā … Uttarā Nandamātā … Visākhā Migāramātā … umat awam perempuan Khujjuttarā … umat awam perempuan Sāmāvatī … Suppavāsā putri Koliya … umat awam perempuan Suppiyā … perumah tangga Nakulamātā …
