56.108–110. Meninggal Dunia sebagai Deva (1-3)

… “Demikian pula, para bhikkhu, hanya sedikit makhluk-makhluk yang, ketika meninggal dunia sebagai deva, terlahir kembali di antara para deva. Tetapi banyak sekali makhluk-makhluk yang, ketika meninggal dunia sebagai deva, terlahir kembali di alam neraka … di alam binatang … di alam setan …”
