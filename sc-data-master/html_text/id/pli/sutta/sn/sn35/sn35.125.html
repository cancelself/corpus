35.125. Di antara Para Vajji

Pada suatu ketika Sang Bhagavā sedang menetap di antara penduduk Vajji di Hatthigāma. Kemudian perumah tangga Ugga dari Hatthigāma mendatangi Sang Bhagavā … dan berkata kepada Beliau …
