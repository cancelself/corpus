35.237. Perumpamaan Tangan dan Kaki (2)

“Para bhikkhu, ketika ada tangan, maka ada memungut dan meletakkan …

“Demikian pula, para bhikkhu, ketika ada mata, maka kesenangan dan kesakitan muncul secara internal dengan kontak-mata sebagai kondisi … Ketika ada pikiran, maka kesenangan dan kesakitan muncul secara internal dengan kontak-pikiran sebagai kondisi.

“Ketika, para bhikkhu, tidak ada tangan, maka tidak ada memungut dan meletakkan …

“Demikian pula, para bhikkhu, ketika tidak ada mata … tidak ada pikiran, maka tidak ada kesenangan dan kesakitan muncul secara internal dengan kontak-pikiran sebagai kondisi.”

Translated by Indra Anggara for Dhamma Citta.
