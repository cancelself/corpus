35.62. Padamnya Segala Kemelekatan (2)

“Para bhikkhu, Aku akan mengajarkan kepada kalian mengenai Dhamma menuju padamnya segala kemelekatan. Dengarkanlah …”

“Dan apakah, para bhikkhu, Dhamma menuju padamnya segala kemelekatan itu? Bagaimana menurut kalian, para bhikkhu, apakah mata adalah kekal atau tidak kekal?”

… Dilengkapi seperti pada §32 … PTS vp Pali 4.35

“Ini, para bhikkhu, adalah Dhamma menuju padamnya segala kemelekatan.”
