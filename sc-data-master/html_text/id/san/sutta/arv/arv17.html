17. Lima Kekuatan

Di sana, para bhikkhu, apakah lima kekuatan?

Mereka adalah:

  1. Kekuatan keyakinan,
  2. Kekuatan semangat,
  3. Kekuatan perhatian,
  4. Kekuatan konsentrasi,
  5. Kekuatan kebijaksanaan.

Ini, para bhikkhu, adalah lima kekuatan.
