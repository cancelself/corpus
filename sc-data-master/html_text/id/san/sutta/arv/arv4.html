4. Dua Belas Landasan Indera

Di sini, para bhikkhu, apakah dua belas landasan indera?

Mereka adalah:

  1. Landasan indera mata internal,
  2. Landasan indera bentuk eksternal,
  3. Landasan indera telinga internal,
  4. Landasan indera suara eksternal,
  5. Landasan indera hidung internal,
  6. Landasan indera bebauan eksternal,
  7. Landasan indera lidah internal,
  8. Landasan indera rasa kecapan eksternal,
  9. Landasan indera badan internal,
  10. Landasan indera sentuhan eksternal,
  11. Landasan indera pikiran internal,
  12. Landasan indera objek pikiran eksternal.

Ini, para bhikkhu, disebut sebagai dua belas landasan indera.
