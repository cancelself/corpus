17. Lima Kekuatan

Disini, para Bhikṣu, apakah lima kekuatan?

Itu adalah:

  1. Kekuatan keyakinan,
  2. Kekuatan kegigihan,
  3. Kekuatan perhatian,
  4. Kekuatan konsentrasi,
  5. Kekuatan kebijaksanaan.

Inilah, para Bhikṣu, lima kekuatan.
