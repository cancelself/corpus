23. Муравейник

SC 1Nya 1Ms 9M_837Msdiv _249Я слышал​, что как-то раз, когда Благословенный пребывал в ашраме Анатхапиндики в Роще Джеты близ Раджагахи, а достопочтенный Кумаракассапа пребывал в Роще слепцов, осветив собой почти всю округу Рощи слепцов, достопочтенного Кумаракассапу посетило некое божество прекрасной наружности и, став в стороне, обратилось к нему:

Иносказание о муравейнике, рассказанное божеством Кумаракассапе

SC 2Nya 2PTS vp Pali 1.143Ms 9M_838«Монах, а монах, ночью этот муравейник дымится, днём же пылает.

Брахман сказал: «Мудрый, копай с помощью ножа.» Копая с помощью ножа, мудрый увидел поперечину: «Поперечина, господин.»

Брахман сказал: «Выбрось поперечину, мудрый, и продолжай копать с помощью ножа.» Продолжив копать с помощью ножа, мудрый увидел жабу: «Жаба, господин.»

Брахман сказал: «Выбрось жабу, мудрый, и продолжай копать с помощью ножа.» Продолжив копать с помощью ножа, мудрый увидел развилку: «Развилка, господин.»

Брахман сказал: «Выбрось развилку, мудрый, и продолжай копать с помощью ножа.» Продолжив копать с помощью ножа, мудрый увидел решето: «Решето, господин.»

Брахман сказал: «Выбрось решето, мудрый, и продолжай копать с помощью ножа.» Продолжив копать с помощью ножа, мудрый увидел черепаху: «Черепаха, господин.»

Брахман сказал: «Выбрось черепаху, мудрый, и продолжай копать с помощью ножа.» Продолжив копать с помощью ножа, мудрый увидел тесак и разделочную колоду: «Тесак и разделочная колода, господин.»

Брахман сказал: «Выбрось тесак и разделочную колоду, мудрый, и продолжай копать с помощью ножа.» Продолжив копать с помощью ножа, мудрый увидел кусок мяса: «Кусок мяса, господин.»

Брахман сказал: «Выбрось кусок мяса, мудрый, и продолжай копать с помощью ножа.» Продолжив копать с помощью ножа, мудрый увидел змия: «Змий, господин.»

Брахман сказал: «Оставь змия, мудрый, не причиняй ему вреда. Почитай змия.»

SC 3Ms 9M_839Монах, обо всём этом тебе следует пойти-расспросить Благословенного и запомнить то, что он ответит. Монах, в мире с его богами, Марами, Брахмами, отшельниками, брахманами, вельможами и простолюдинами я не вижу того, кто мог бы, дав ответ, удовлетворить пытливость ума, кроме Так Приходящего, учеников Так Приходящего или тех, кто слышал ответ от них.»

Так сказало божество и тотчас исчезло.

Просьба Кумаракассапы к Будде разъяснить образы иносказания

SC 4Nya 3Ms 9M_840Msdiv _250Тогда, по прошествии ночи, достопочтенный Кумаракассапа отправился к Благословенному. По прибытии он, почтительно его поприветствовав, сел в стороне. Сев в стороне, он [рассказал Благословенному о произошедшем и добавил]: «Преподобный, что означает муравейник, что означает дымится ночью, пылает днём, кто такой брахман, кто мудрый, что означают нож, копать, поперечина, жаба, развилка, решето, черепаха, тесак и разделочная колода, кусок мяса и змий?»

Разъяснение Будды

SC 5Nya 4PTS vp Pali 1.144Ms 9M_841Msdiv _251«Монах, муравейник есть название для тела, состоящего из четырёх элементов, рождённого от отца с матерью, вскормленного рисом и бобами, подверженного износу, тлену распаду и разрушению.

SC 6Ms 9M_842О том, что совершается днём, ночью думают и размышляют. Это и есть «дымится ночью». То, о чём ночью подумали и поразмыслили, совершают днём посредством тела, речи и ума. Это и есть «пылает днём». (2–3.)

SC 7Ms 9M_843Брахман есть название для Так Приходящего, Архата, Совершенно пробуждёного. Мудрый есть название для монаха-адепта. (4–5.)

SC 8Ms 9M_844Нож есть мудрость благородных. Копать означает проявлять усердие. (6–7.)

SC 9Ms 9M_845Поперечина есть название для неведения. Выбрось поперечину - устрани неведение, мудрый, и продолжай копать с помощью ножа - таков смысл.

SC 10Ms 9M_846Жаба есть название для взбудораженности гневом. Выбрось жабу - устрани взбудораженность гневом, мудрый, и продолжай копать с помощью ножа - таков смысл.

SC 11Ms 9M_847Развилка есть название для сомнения. Выбрось развилку - устрани сомнения, мудрый, и продолжай копать с помощью ножа - таков смысл.

SC 12Ms 9M_848Решето есть название для пяти умственных помех, а именно помехи чувственного желания, помехи недоброжелательности, помехи вялости и оцепенения, помехи беспокойства и самоукорения, помехи сомнения. Выбрось решето - устрани пять умственных помех, мудрый, и продолжай копать с помощью ножа - таков смысл.

SC 13Ms 9M_849Черепаха есть название для пяти агрегатов цепляния, а именно агрегат цепляния за форму, агрегат цепляния за чувство, агрегат цепляния за восприятие, агрегат цепляния за задатки, агрегат цепляния за сознание. Выбрось черепаху - устрани пять агрегатов цепляния, мудрый, и продолжай копать с помощью ножа - таков смысл.

SC 14PTS vp Pali 1.145Ms 9M_850Тесак и разделочная колода есть название для пяти струн чувственности - формы, воспринимаемые глазом, вожделенные, желанные, дорогие, приятные, связанные с чувственными удовольствиями, вызывающие страсть, звуки, воспринимаемые ухом... запахи, воспринимаемые носом... вкусы, воспринимаемые языком... прикосновения, воспринимаемые телом, вожделенные, желанные, дорогие, приятные, связанные с чувственными удовольствиями, вызывающие страсть. Выбрось тесак и разделочную колоду - выбрось пять струн чувственности мудрый, и продолжай копать с помощью ножа - таков смысл.

SC 15Ms 9M_851Кусок мяса есть название для страсти к наслаждениям. Выбрось кусок мяса - устрани страсть к наслаждениям, мудрый, и продолжай копать с помощью ножа - таков смысл.

SC 16Ms 9M_852Змий есть название для уничтожившего загрязнения. Таков смысл «Оставь змия, мудрый, не причиняй ему вреда. Почитай змия.»

SC 17Ms 9M_853Так моливл Благословенный, и достопочтенный Кумаракассапа возрадовался его словам.
