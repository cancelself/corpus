4.146. Время (I)

Благословенный сказал: «Монахи, есть эти четыре времени. Какие четыре?

  1. время для слушания Дхаммы,
  2. время для обсуждения Дхаммы,
  3. вредя для успокоения,
  4. время для прозрения.

Таковы четыре времени».
