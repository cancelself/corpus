6.113. Недовольство

Благословенный сказал: «Монахи, есть эти три вещи. Какие три?

  1. недовольство,
  2. причинение вреда,
  3. поведение, противоположное Дхамме.

Таковы три вещи. Три иные вещи следует развить ради отбрасывания этих трёх вещей. Какие три?

  1. Сорадование следует развить для отбрасывания недовольства.
  2. Безвредность следует развить для отбрасывания причинения вреда.
  3. Поведение в соответствии с Дхаммой следует развить для отбрасывания поведения, противоположного Дхамме.

Эти три вещи следует развить ради отбрасывания первых трёх вещей».
