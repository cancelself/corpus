• Colección de discursos agrupados numéricamente

1.188–197. Primer sub-capítulo

“Monjes, el primero entre mis discípulos monjes en el señorío es Annakondanna.

“Monjes, el primero entre aquellos de gran sabiduría es Sariputta.

“Monjes, el primero entre aquellos con poderes espirituales es Mahamoggallana.

“Monjes, el primero entre aquellos que expusieron las prácticas ascéticas es Mahakassapa.

“Monjes, el primero entre aquellos con el ojo divino es Anuruddha.

“Monjes, el primero entre aquellos de familias eminentes es Bhaddiya Kaligodhayaputta.

“Monjes, el primero entre aquellos con dulce voz es Lakuntaka Bhaddiya.

“Monjes, el primero entre aquellos con el rugido de león es Pindola Bharadvaja.

“Monjes, el primero entre aquellos que habla sobre el Dhamma es Punna Mantaniputta.

“Monjes, el primero entre aquellos que explican en detalle el significado de lo que ha sido dicho en forma breve es Mahakaccana.
